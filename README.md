# Banking System (Java)

A simple console-based Banking System application written in Java. This project demonstrates basic Java concepts such as classes, methods, loops, conditionals, user input handling, and encapsulation.

## Description

This banking system allows a user to:
- Enter their name
- View their account balance
- Deposit money
- Withdraw money
- Exit the application

The program runs in a loop until the user chooses to exit. The initial account balance is set to 1000.

## Features

- User name input
- Menu-driven console interface
- Show current balance
- Deposit money into the account
- Withdraw money with balance validation
- Continuous operation until exit
- Written entirely using core Java

## Technologies Used

- Java (JDK 8 or higher recommended)
- Java Standard Library
- Scanner class for user input
